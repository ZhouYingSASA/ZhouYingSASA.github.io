官方 Windows Preinstallation Environment (WinPE) 安装
2019-05-12

需要安装Deployment Tools：

  • adksetup
    • Deployment Tools

无图形界面winpe的安装软件：

  • adkwinpesetup

 

（文中x为安装win的版本，如10）
（文中..\为安装目录）

Windows ADK needs extra environment settings.
ADK安装对环境变量有额外需求，下文描述对其中几个命令的更改。
######它看起来就像一个未完成品

WinPE的copype文件对环境变量有所依赖，所以需要更改环境变量配置。

ERROR: The following processor architecture was not found: amd64.
Failed!

进入..\Windows Kits\x\Assessment and Deployment Kit\Windows Preinstallation Environment
修改copype.cmd文件

添加以下环境变量

即可执行copype { amd64 | x86 | arm | arm64 } <workingDirectory>

但是你发现MakeWinPEMedia还是不能执行:

'oscdimg' 不是内部或外部命令，也不是可运行的程序或批处理文件。
ERROR: Failed to create ".." file.

原因还是缺少了oscdimg的路径变量，这里我直接把oscdimg设定成一个常量，
值为它所在的位置了。

而后执行MakeWinPEMedia {/ufd | /iso} [/f] <workingDirectory> <destination>
即可生成相应刻录文件。
